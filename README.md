# Bondi Template

**Bondi** is a modern and fully responsive template designed for creative agencies. Built with Bootstrap, this template features a clean and well-structured sections, smooth typography, and a professional layout.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Sticky navigation header with search icon
- Services showcase (Graphic Design, Web Design, App Design)
- Interactive portfolio grid with category filtering
- About section with company information
- Team member profiles with descriptions
- Blog section with featured articles
- Newsletter subscription banner
- Footer with links and social media icons
- Smooth animations and transitions

## Technologies Used

- HTML5
- CSS3 (With Font Awesome icons)
- Bootstrap (For layout and responsiveness)
- Google Fonts (Roboto)

## Installation and Usage

1. Clone or download this repository to your local machine.
2. Open `index.html` in any modern web browser to view the template.
3. Customize the content in `index.html` and styles in `css/bondi.css` as needed.
4. Update images in the `images/` folder with your own content.
5. Modify the JavaScript in `js/` files for custom functionality.
6. For deployment, upload all files to your web server.
